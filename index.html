About Me

A web developer with a designer’s mindset—passionate about clean UIs, Gestalt principles, and exploring tomorrow’s tech today.
I’m a full-stack web application developer who believes great code should be matched by even better design. My approach blends technical precision with a deep understanding of UI/UX fundamentals, especially Gestalt principles like proximity, similarity, and continuity—ensuring every interface I build feels intuitive and harmonious.
